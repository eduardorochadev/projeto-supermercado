# Frontend

Esta pasta conterá o projeto Angular do sistema de supermercado.

## Para criar o projeto Angular:

```bash
cd frontend
npx @angular/cli@latest new projeto-frontend --directory=. --routing --style=css
```

## Estrutura planejada:
- Componentes para CRUD de produtos
- Serviços para consumir a API REST
- Roteamento entre páginas
- Interface responsiva
